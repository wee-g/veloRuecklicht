veloRuecklicht
==============
 Code fuer Velo Ruecklicht
 
 - Modus 1 = Lauflicht
 - Modus 2 = 5 LED High

ATtiny 85
5 LEDs an 5 Pins mit 220Ohm Widerstand (5V)
Schiebeschalter mit Pulldown Widerstand an Pin 5
